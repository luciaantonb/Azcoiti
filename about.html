Pattern Creation

Azcoiti is a generative system that translates data into woven textile patterns. Numbers become structure. Structure becomes fabric.

About

Why

The fashion industry produces numbers that are difficult to feel. Millions of tonnes of waste, trend cycles that once lasted decades now lasting weeks. The scale makes them abstract, and abstraction makes them easy to ignore. This project is an attempt to make that data material.

How

It was built on weave draft logic, the same mathematics used to program a loom. A dataset becomes a weave draft. Numbers become the density, direction, and variation of a textile surface. Each artwork is then produced as a physical object, because to understand the data it should be something you can touch.

What

Azcoiti is a generative system that translates any dataset into woven textile patterns, where every visual property is determined by a real number. The tool is open to anyone. The five artworks here apply it to fast fashion research, not because it is the only valid subject, but because it is the one that made this project feel necessary.

Which

Of all the elements in a garment, pattern is the only one with no structural purpose. It exists purely to express something. It is the most human part of fashion, the part that has nothing to do with function and everything to do with meaning. Encoding data into pattern felt like the most honest translation: asking decoration to carry weight again.
